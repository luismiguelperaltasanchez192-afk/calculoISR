# Calculadora de ISR - República Dominicana

## Descripción

Este proyecto consiste en el desarrollo de un programa en C++ que calcula el Impuesto Sobre la Renta(ISR) de un empleado, utilizando la escala salarial vigente de la Dirección General de Impuestos Internos(DGII) de la República Dominicana.

El programa solicita al usuario el sueldo mensual, calcula el sueldo anual, determina si aplica el ISR y muestra el monto del impuesto correspondiente.Si el empleado no debe pagar ISR, el programa muestra un descuento de RD$0.00.

## Objetivos

- Leer el sueldo mensual de un empleado.
- Calcular el sueldo anual.
- Determinar el ISR según la escala de la DGII.
- Mostrar el sueldo mensual, sueldo anual, ISR anual e ISR mensual.
- Indicar cuando el empleado está exento del pago de ISR.

## Tecnologías utilizadas

- Lenguaje: C++
- Entorno de desarrollo : Visual Studio
- Sistema operativo : Windows

## Funcionamiento

1. El usuario introduce el sueldo mensual.
2. El programa calcula el sueldo anual.
3. Se compara el sueldo anual con la tabla de la DGII.
4. Se calcula el ISR correspondiente.
5. Se muestran los resultados en pantalla.
